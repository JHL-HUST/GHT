# GHT

This is the code for our EMNLP 2022 paper **Graph Hawkes Transformer for Extrapolated Reasoning on Temporal Knowledge Graphs**

![TITer](./img/GHT.png)

### Train and Test

you can run as following:

```
python3 main.py --do_train --do_test
```
